# Чеклист перед объявлением тестирования

Краткий список того, что проверить до того, как сообщить пользователям о старте тестов.

**Пошаговое выполнение с командами:** [RUNBOOK_BEFORE_TESTING.md](./RUNBOOK_BEFORE_TESTING.md) — делайте всё по шагам по порядку.

## 1. Приложение на ВМ запущено и отвечает

- [ ] **Health:** `curl -s -o /dev/null -w "%{http_code}" https://ваш-домен/api/health` → **200**
- [ ] Главная страница открывается в браузере
- [ ] Если 502 — на ВМ не запущен Node или он упал. Перезапуск: после деплоя через `systemctl restart domeo-standalone` или скрипт деплоя; не использовать только `restart-vm-app.ps1` без systemd (см. [VM_502_DIAGNOSTICS.md](./VM_502_DIAGNOSTICS.md))

## 2. Деплой и каталог на ВМ

- [ ] На ВМ в **~/domeo-app** (или ваш REMOTE_PATH) лежит **standalone**: есть `server.js`, `node_modules`, `public`, `.next` — результат деплоя артефактом, а не «голый» sync исходников
- [ ] Если деплой делали в ~/domeo-app, перезапуск должен идти из ~/domeo-app (`node server.js`), не из ~/1002doors

## 3. Переменные окружения на ВМ

- [ ] В рабочей папке приложения есть **.env** с минимум: **DATABASE_URL**, **JWT_SECRET** (не короче 32 символов)
- [ ] Для PDF на ВМ: **PUPPETEER_EXECUTABLE_PATH** (или Chromium установлен в стандартный путь), иначе экспорт PDF выдаст ошибку про браузер

## 4. Фото (ручки, двери, цвета)

- [ ] **Синхронизация uploads на ВМ выполнена** (например `.\scripts\sync-uploads-to-vm.ps1` или аналог), иначе фото цветов/ручек могут не находиться → 502 или пустые квадраты
- [ ] В `public/uploads/final-filled/` на ВМ есть ожидаемые папки: ручки (04_Ручки_Завертки), doors, Наличники — имена как в [STAGING_READY.md](./STAGING_READY.md)

## 5. Память и стабильность

- [ ] На ВМ добавлен **swap** (например 2 GB), чтобы при пиках (несколько экспортов PDF, много пользователей) не было OOM и 502. Скрипт: `.\scripts\vm-add-swap.ps1`
- [ ] Очередь PDF включена (по умолчанию до 2 одновременных экспортов); при большом числе запросов часть пользователей получит «Сервер занят экспортом PDF. Попробуйте через минуту» — это ожидаемо

## 6. Nginx (если используется)

- [ ] Конфиг применён: `.\scripts\apply-nginx-to-vm.ps1` (таймауты для /uploads/, лимиты)
- [ ] В конфиге `root` для `/uploads/` указывает на каталог приложения (например `/home/ubuntu/domeo-app/public`)

## 7. Что сказать тестировщикам

- Тестируем на **staging / тестовом окружении**; возможны кратковременные сбои — пишите, что именно увидели и при каком действии
- При экспорте PDF: если много людей экспортируют одновременно, возможна очередь (ожидание до нескольких минут или сообщение «Попробуйте через минуту»)
- Если не загружаются фото цветов/ручек — мы проверим синхронизацию фото и пути на сервере (см. [PHOTO_COLOR_502_ANALYSIS.md](./PHOTO_COLOR_502_ANALYSIS.md))
